Introduction to Python

Overview

Python is a high-level, interpreted programming language known for its simplicity and readability. It supports multiple programming paradigms, including procedural, object-oriented, and functional programming. Python is widely used in web development, data analysis, artificial intelligence, scientific computing, and more.

Installation

To install Python, follow these steps:

  1. Go to the official Python website: python.org
  2. Download the latest version of Python for your operating system.
  3. Run the installer and follow the instructions.
  4. Verify the installation by opening a terminal or command prompt and typing python --version.

Basic Syntax

Python syntax is designed to be readable and straightforward. Here are a few key points:

  • Python uses indentation to define code blocks.
  • Statements end with a newline, not a semicolon.
  • Comments are written with a hash symbol (#).

                # This is a comment
                print("Hello, World!")  # Output: Hello, World!

Variables

Variables in Python are created when you assign a value to them. You do not need to declare the type of a variable explicitly.


                x = 5
                y = "Hello, World!"
                print(x)  # Output: 5
                print(y)  # Output: Hello, World!

Data Types

Python has several built-in data types, including:

  • int: Integer numbers
  • float: Floating-point numbers
  • str: Strings
  • list: Ordered collections of items
  • dict: Key-value pairs

            # Examples of data types
            a = 10        # int
            b = 3.14      # float
            c = "Python"  # str
            d = [1, 2, 3] # list
            e = {"key": "value"}  # dict

            print(type(a))  # Output:

                    # Add two numbers
